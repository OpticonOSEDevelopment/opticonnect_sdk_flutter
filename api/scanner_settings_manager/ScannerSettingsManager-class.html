ScannerSettingsManager

ScannerSettingsManager class

A class representing the scanner settings for Opticon BLE scanners.

This class provides access to various settings, including enabling specific codes, configuring scan options, and managing formatting options.

It is recommended that you access this class through the OptiConnectSDK and avoid initializing or modifying the settings manually.

Example usage:

final sdk = OptiConnectSDK.instance;
sdk.initialize();
final scannerSettings = sdk.scannerSettings;
