CodeSpecific

CodeSpecific class abstract

Interface for accessing settings for specific barcode symbologies.

This interface includes settings for individual barcode types such as Codabar, Code 39, UPC, EAN, and others, allowing configuration of scanner behavior for specific barcode formats.
